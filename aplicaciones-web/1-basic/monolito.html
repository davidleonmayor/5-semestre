Evolucion de los sistemas computacionales
Sistemas Concepto Imagen
Monolitico Los sistemas monolíticos son aquellos en los que su centro es un grupo de estructuras fijas, las cuales funcionan entre sí. En los sistemas monolíticos existen módulos grandes en el núcleo, los cuales interactúan entre sí, para poder tener esta estructura, las diferentes partes del kernel son compiladas por capas.
Microservicio Enfoque arquitectónico para el desarrollo de software Los microservicios son un enfoque arquitectónico para el desarrollo de software en el que una aplicación se compone de pequeños servicios independientes que se comunican entre sí a través de APIs bien definidas. Cada microservicio se puede implementar y ejecutar de forma independiente, lo que permite una mayor flexibilidad y escalabilidad en el desarrollo de aplicaciones. Este enfoque permite que equipos pequeños y autónomos trabajen en diferentes servicios sin depender de otros, facilitando así el desarrollo y mantenimiento del software.
